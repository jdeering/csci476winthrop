LoginGUI Namespace

Namespace hierarchy

Classes

Class Description
AdminForm AdminForm contains the user interface for the Administration Panel, which includes password reset and report generation functions.
ChartForm ChartForm contains the user interface for displaying reports generated in the AdminForm.
Login Login contains methods for storing and looking up usernames and passwords, as well as an MD5 encryption method.
LoginForm LoginForm contains a user interface for logging in and create user functions.
UserStats UserStats contains methods to parse user and module score XML files, as well as the structures to contain the data of these files.

Structures

Structure Description
UserStats.ModuleScoreData ModuleScoreData holds the module name, a list of ModuleScoreNodes, and the size of this list.
UserStats.ModuleScoreNode ModuleScoreNodes hold the username and score of an individual score.
UserStats.UserScoreData UserScoreData holds the username, module, last score, best score, first score, average score, list of UserScoreNodes, and the list size of the recent scores.
UserStats.UserScoreNode UserScoreNodes hold the date, time and score of a single user score.
